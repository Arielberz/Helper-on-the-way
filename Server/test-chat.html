1 Connect to Server

How to get a token:
1. Use Postman or cURL to login: POST http://localhost:3001/api/users/login
2. Copy the token from the response
3. Paste it above and click Connect

2 Join Conversation

How to get a conversation ID:
1. Create a request (if you haven't): POST /api/requests
2. Assign a helper to it: PATCH /api/requests/:id
3. Get conversation: GET /api/chat/conversation/request/:requestId
4. Copy the conversation _id from response
